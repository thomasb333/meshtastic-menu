# Meshtastic Programmer's Reference Guide

> Line-by-line configuration reference for Meshtastic firmware **2.7+**  
> Covers all Radio Config, Module Config, app settings, CLI examples, and detailed enum value definitions.


## 📱 Phone Access

Open the URL in any mobile browser. The guide is fully responsive:

- **Bottom nav bar** — tap ☰ Menu to browse all sections
- **Platform switcher** — toggle Android / iOS / Web+CLI
- **Search** — tap ⌕ or use the search bar at the top
- **Tap any setting** to expand full details
- **Tap dashed enum pills** (like role values) for detailed definitions

Works in Safari (iOS), Chrome (Android), and any modern mobile browser. No app install needed.

## 📡 About Meshtastic

[Meshtastic](https://meshtastic.org) is an open-source, off-grid mesh communication platform using LoRa radios. This guide is community-maintained and unofficial. Always verify against the [official documentation](https://meshtastic.org/docs/configuration/).
